# nova-business

Nova（孙子霖）的生意与内容项目仓库（**公开 / public**）。

- 用途：存放与生意、内容项目相关的资料、脚本、文档
- 维护方式：由 WorkBuddy 协助初始化与管理
- 连接方式：SSH（已验证可用）

## 仓库地址
- HTTPS：`https://github.com/NOVASUN168/nova-business.git`
- SSH：`git@github.com:NOVASUN168/nova-business.git`

## 本仓库包含
- `README.md`：本说明文件
- `GitHub账号信息.md`：你的 GitHub 账户与连接信息汇总（方便以后随时查看）

## 常用命令（在终端里用）
```bash
# 首次把仓库下载到本地
git clone git@github.com:NOVASUN168/nova-business.git

# 查看当前改动状态
git status

# 把改动加入暂存区并提交
git add .
git commit -m "描述这次改了什么"

# 推送到 GitHub
git push

# 从 GitHub 拉取最新内容
git pull
```
